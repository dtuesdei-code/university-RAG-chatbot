# University RAG Chatbot

A student assistant chatbot built with Retrieval-Augmented Generation (RAG), answering questions about regulations, curriculum, and by-laws using official university documents.

## 🚀 Demo

👉 [Test the Chatbot](https://university-rag-chatbot.streamlit.app)

## ✨ Features

- Answers student queries using RAG
- Built with Python + LangChain
- Uses ChromaDB for vector storage
- Integrates OpenAI models for natural language responses
- Streamlit UI for easy interaction

## 📦 Installation

Clone the repo and install dependencies:

```bash
git clone https://github.com/dtuesdei-code/university-RAG-chatbot.git
cd university-RAG-chatbot
pip install -r requirements.txt
```
